Concrete (Regression)
---
* Download the following files from the [UCI Repository: Concrete](https://archive.ics.uci.edu/ml/datasets/Concrete+Compressive+Strength) to this directory.
    * `Concrete_Data.xls`.

* Preprocess the data.
    * Run `python3 preprocess.py`.
    	* This creates a `data.npy`.
