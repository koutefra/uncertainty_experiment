Wine (Regression)
---
* Download the following files from the [UCI Repository: Wine](https://archive.ics.uci.edu/ml/datasets/Wine+Quality) to this directory.
    * `winequality-red.csv`.
    * `winequality-white.csv`.

* Preprocess the data.
    * Run `python3 preprocess.py`.
    	* This creates a `data.npy`.
